# Process Documentation

Thư mục này ghi lại implementation thực tế sau khi từng phase được triển khai.

Quy ước:

- Mỗi phase có một file riêng, ví dụ `phase-0-foundation.md`, `phase-1-auth.md`.
- File phase bên ngoài `.notes/phase-*.md` là plan/yêu cầu; file trong thư mục này là trạng thái đã cài đặt.
- Khi hoàn thành thêm một phase, cập nhật file process tương ứng trước khi coi phase đó xong.
